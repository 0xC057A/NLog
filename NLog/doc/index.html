NLog - A .NET Logging Tool

What is NLog?

NLog is a simple logging library similar in concepts to log4net, designed with simplicity and flexibility in mind. NLog project is hosted on sourceforge.net.

Features

  • .NET, .NET Compact Framework and Mono 1.0 support
  • very easy to configure, both with a config file and programmatically
  • easy to use - interface is identical to Console.WriteLine

News

2004-08-23 Initial support for filter infrastructure added to SVN repository. NLog core is feature complete now. Future work will concentrate on stabilizing things and adding new appenders, layout appenders and filters.

2004-08-21 Public announcement of the NLog project.

Current Status

NLog has not been released yet, but the source code is available for testing. It's open source, licensed under the terms of BSD license, which permits commercial use.

NLog can be downloaded from the Subversion repository. Subversion write access is available on request. See the repository page for details. For your convenience the daily subversion snapshots are available here.

Binary releases will be available after I get some feedback from the community and the API is stable enough.

Building NLog

NLog can be built with either Visual Studio.NET 2003 or NAnt. If you decide to use NAnt - make sure to use one of the latest nightly builds available at http://nant.sourceforge.net/nightly/builds/.

Mailing List

If you have any NLog-related questions or suggestions, join the NLog mailing list: nlog-list@lists.sourceforge.net and post your questions there.
