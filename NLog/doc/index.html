NLog - A .NET Logging Tool

What is NLog?

NLog is a .NET logging library similar in concepts to log4net, designed with simplicity and flexibility in mind. NLog project is hosted at sourceforge.net.

Features

  • .NET, .NET Compact Framework and Mono 1.0 support
  • very easy to configure, both with a config file and programmatically
  • easy to use - interface is identical to Console.WriteLine

Current Status

NLog 0.2 beta1 has been released in source and binary form. It's open source, licensed under the terms of BSD license, which permits commercial use. Everyone is encouraged to test it and report feedback to the mailing list.

The logging API is really minimal and shouldn't change in the future. Configuration API isn't ready yet, but the configuration file support should be pretty usable. The support for .NET CF is in subversion but no packages are available yet. They will be, when it receives some testing and positive feedback from the community.

Downloading NLog

NLog releases can be downloaded from the sourceforge file release system. NLog can be also downloaded from the Subversion repository. Subversion write access is available on request. See the repository page for details.

For your convenience the source and binary snapshots (including the windows installer) are available here.

Building NLog

NLog can be built with either Visual Studio.NET 2003 or NAnt. If you decide to use NAnt - make sure to use one of the latest nightly builds available at http://nant.sourceforge.net/nightly/builds/.

Mailing List

If you have any NLog-related questions or suggestions, join the NLog mailing list: nlog-list@lists.sourceforge.net and post your questions there.
