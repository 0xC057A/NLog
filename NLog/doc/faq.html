What is NLog?

NLog is a simple logging library similar in concepts to log4net, designed with simplicity and flexibility in mind.

Why should I use NLog? Why not log4net?

There are many logging libraries out there, log4net being one of them.

Most people I know have problems using log4net because it's difficult configure properly. Log4net's usage is also complicated because its logging interface doesn't take advantage of .NET features like functions with variable number of arguments. Also log4net is not very portable - there are many framework-specific versions of the library (one for .NET 1.0, onet for .NET 1.1, one for Mono, and so on) which makes binary portability difficult.

NLog is an answer to all these shortcomings: it's easy to configure, easy to use, easy to extend and portable across frameworks.
